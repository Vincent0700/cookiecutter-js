# JavaScript 基础模板创建工具

## Feature

1. 基础配置文件生成
    - .gitignore
    - package.json
    - LICENSE
    - README.md
    
2. eslint 相关配置文件生成
    - .eslintrc.js
    - .eslintignore
    - .prettierrc.js
    - .editorconfig
    - .vscode/settings.json

3. 生成完毕后执行 yarn install

## Usage

```bash
$ cookiecutter gh:vincent0700/cookiecutter-js
```
